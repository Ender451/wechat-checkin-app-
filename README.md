# 打卡小程序

一个功能完整的微信小程序，用于记录和管理工作时间。

## 功能特性

- 🕐 **实时打卡**：支持上下班打卡，精确到秒
- ⏱️ **工作时长计算**：自动计算每日工作时长
- 📊 **数据统计**：提供详细的工作数据分析和图表展示
- 📝 **记录管理**：支持查看、编辑和删除打卡记录
- 💾 **本地存储**：数据安全存储在本地
- 🎨 **美观界面**：现代化的UI设计和流畅的用户体验

## 主要功能

### 打卡功能
- 上班打卡：记录开始工作时间
- 下班打卡：记录结束工作时间
- 支持重复更新下班打卡时间
- 实时显示当前工作时长

### 记录管理
- 查看历史打卡记录
- 编辑打卡时间和备注
- 删除错误的打卡记录
- 按日期筛选和排序

### 数据统计
- 工作时长统计分析
- 按日、周、月统计
- 图表化数据展示
- 导出统计报告

## 技术栈

- **前端框架**：微信小程序原生开发
- **数据存储**：微信小程序本地存储 API
- **UI组件**：自定义组件开发
- **图表库**：Canvas 绘图

## 开发环境

- 微信开发者工具
- Node.js (可选，用于依赖管理)
